# 🎓 Student Management System

A simple Student Management System built using Node.js, Express, MongoDB, and EJS.  
This project performs basic CRUD operations with image upload and flash message support.

---

## 🚀 Features

- Add new student
- View all students
- Update student details
- Delete student record
- Image upload (profile photo)
- Flash messages for success & error
- Clean UI using Bootstrap

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB & Mongoose
- EJS Templates
- Bootstrap 5
- Multer (File Upload)
- Express-session & connect-flash
